Game Development Club

2022 Drawtober Contest

This is a collection of all my submissions for my dorm's "Drawtober" contest, which we won! We were tasked with drawing something that followed the Halloween-themed prompt for each day. This only includes a few days of October, as I split the work with my other three roommates. 

Low-Polygon Gecko

This was my first project in my Digital 3D class where I utilized blender to recreate the geckos from one of my favorite video games: "Fallout: New Vegas." The project had us make an organic form using as few polygons as possible. We were then instructed to apply materials to the form, as well as create an environment for it to inhabit. 

Useless Flip Flop

This was another project for my Digital 3D class where we were instructed to create a virtual sculpture in Blender of an everyday utilitarian object that has been modified to make complicate or remove its function. I went with a flip flop that lacked the straps the keep it attached to the wearer's foot. All that's left is the popsocket-like protrusion where the straps would have started at. This leaves users having to walk awkwardly as the try and grip onto the small handle of the now useless flipflop.

Shape Primitive Art

This was my first project for my "Computer Programming for Creatives" class (CTK 302) where I utilized JavaScript to create a scene that changes when clicked only using shape primitives. I mostly utilized ellipses, lines, and quads and manually plotted the points to form the image I wanted.

Froggy Chase Minigame

Another project from my CTK 302 class where I used JavaScript to create a minigame. The idea is that the player is a frog with the objective of "eating" flies and avoiding the birds. I focused on generating the different stage screens, hit boxes, and enemy/objective capture numbers. The assets used were all drawn by me! Here's a link to play!
